github+hexo搭建个人博客网站

2019-02-28 17:13:31 | nodejs


  很早之前就想做个博客了,最早搭博客是用 阿里云服务器+WordPress,但是毕竟阿里云是要钱的,于是找到了这款hexo,
基于Node.js的静态博客框架,可以方便的生成静态网页托管在GitHub和Coding上,主要原因还是免费！

hexo 官方文档:  https://hexo.io/zh-cn/docs/
[CSDN] hexo 史上最全搭建教程:   https://blog.csdn.net/sinat_37781304/article/details/82729029

废话不多说,直接上步骤：

配置本地环境

git

  windows 下载地址：https://gitforwindows.org/
  安装完成后

git --version

 确认安装是否成功

nodejs

  windows 下载地址：https://nodejs.org/en/download/
  安装完成后

node -v && npm -v

 确认安装是否成功

hexo

  cmd输入命令行：

npm install -g hexo-cli

 安装完成后

hexo -v

 确认安装是否成功

hexo 基本命令

 新建一个blog项目

hexo init blogname
cd blogname
npm install

 生成文件夹目录如下

.
├── _config.yml         // 博客的配置文件**
├── package.json        // 应用程序的信息
├── scaffolds           // 文章模板
├── source              
|   ├── _drafts         // 草稿箱
|   └── _posts          // 发布的博文.md文件目录
└── themes              // 主题文件

 常用基本操作

hexo new "mdname"              // 新建文章
hexo clean                     // 清除缓存
hexo s                         // 启动本地服务预览
hexo g                         // 生成静态页面
hexo d                         // 部署到网站

 

hexo 静态页面 托管到 github
